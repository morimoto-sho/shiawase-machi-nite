# 幸せまちにて

> 眠ると、七十年前の夏にいる。

タイムリープ × 地方青春 × 民俗ホラーの長編小説。

## あらすじ

受験に失敗した朝倉湊は、地方の小さな町へ越してきた。町の名は、幸せ町。もとは「川下町」と呼ばれていた場所だった。

ある夜から、湊は眠るたびに七十年前の同じ町にいた。過去の廃校の屋上で出会った少女・澪。現代で距離を詰めてくる歴史サークルの夏希。

ふたつの時代、ふたりの少女。
この町が忘れようとした、ある夏の出来事を、湊は知ることになる。
